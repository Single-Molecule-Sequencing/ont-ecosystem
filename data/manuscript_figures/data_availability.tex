\section*{Data Availability}

The experiment registry, analysis code, and manuscript figures are available at \url{https://github.com/Single-Molecule-Sequencing/ont-ecosystem}. The registry is provided in YAML format (\texttt{experiments.yaml}) with CSV and JSON exports for compatibility. Public ONT sequencing data was accessed from the ONT Open Data repository at \url{s3://ont-open-data/}. Institutional sequencing data underlying the registry is available upon reasonable request subject to institutional data sharing agreements.
