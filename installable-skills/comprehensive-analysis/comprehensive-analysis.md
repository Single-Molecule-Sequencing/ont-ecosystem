# /comprehensive-analysis

Comprehensive ONT sequencing analysis with KDE distributions, end-reason overlays, runtime estimation, and publication-quality figures. Supports data sampling for quick previews with estimated full runtime..

## Usage

$ARGUMENTS

## Quick Start

```bash
# Basic usage
python skills/comprehensive-analysis/scripts/comprehensive_analysis.py [input]

# With provenance tracking
ont_experiments.py run comprehensive_analysis exp-001 [options]
```

## Options

See `/comprehensive-analysis --help` for all available options.

## Installation

```bash
# Clone repository
git clone https://github.com/Single-Molecule-Sequencing/ont-ecosystem.git

# Install to Claude commands
cp ont-ecosystem/installable-skills/comprehensive-analysis/comprehensive-analysis.md ~/.claude/commands/
```
